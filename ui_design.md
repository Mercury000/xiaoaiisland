# 岛B UI组件设计方案

## 1. 当前问题
- 岛B采用文本组件，{计时}变量直接明文显示
- 缺少样式选择功能
- 缺少等宽数字文本组件支持

## 2. 新设计方案

### 2.1 样式选择器
每个阶段（上课前/上课中/下课后）添加：
- **样式选择**：默认样式 / 等宽数字样式

### 2.2 默认样式（图文组件1 + 文本组件）
- 岛A：图文组件1（图标+文字）
- 岛B：文本组件（纯文字）
- 支持变量：{课名} {开始} {结束} {教室}

### 2.3 等宽数字样式（图文组件1 + 等宽数字文本组件）
- 岛A：图文组件1（图标+文字）
- 岛B：等宽数字文本组件（计时器+数字）
- 支持变量：
  - {计时}：根据阶段自动适配
    - 上课前：倒计时（距离上课时间）
    - 上课中：倒计时（距离下课时间）
    - 下课后：正计时（已下课时间）
  - {正计时}：仅上课中可用（已上课时间）
  - {倒计时}：仅上课中可用（距离下课时间）
  - {开始}：开始时间
  - {结束}：结束时间
- 后置小字：可自定义

### 2.4 变量替换逻辑

#### 上课前（STATE_COUNTDOWN）
- {计时}：倒计时（距离上课时间）- 自动适配
- {正计时}：不适用
- {倒计时}：不适用
- 默认值：{计时} + "后上课"

#### 上课中（STATE_ELAPSED）
- {计时}：倒计时（距离下课时间）- 自动适配
- {正计时}：正计时（已上课时间）
- {倒计时}：倒计时（距离下课时间）
- 默认值：{计时} + "后下课"

#### 下课后（STATE_FINISHED）
- {计时}：正计时（已下课时间）- 自动适配
- {正计时}：不适用
- {倒计时}：不适用
- 默认值：{计时} + "已下课"

## 3. UI布局结构

### 3.1 岛B输入区域（每个阶段）
```
样式选择：[默认样式] [等宽数字样式]

如果选择默认样式：
  岛B（右侧文字）：[输入框]

如果选择等宽数字样式：
  计时变量：[下拉选择 {计时}/{正计时}/{倒计时}/{开始}/{结束}]
  后置小字：[输入框]
```

## 4. 数据存储

### 4.1 SharedPreferences结构
```
tpl_b_style_pre: "default" 或 "digital"
tpl_b_style_active: "default" 或 "digital"
tpl_b_style_post: "default" 或 "digital"

tpl_b_timer_var_pre: "{计时}" 或 "{开始}" 或 "{结束}"
tpl_b_timer_var_active: "{计时}" 或 "{正计时}" 或 "{倒计时}" 或 "{开始}" 或 "{结束}"
tpl_b_timer_var_post: "{计时}" 或 "{开始}" 或 "{结束}"

tpl_b_suffix_pre: "后上课"  // 后置小字
tpl_b_suffix_active: "后下课"
tpl_b_suffix_post: "已下课"
```
